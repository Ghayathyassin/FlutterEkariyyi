01 — DIRECTION & SIGNATURE

Quiet document, one bold survey line

The registry's real material is survey linework — parcel boundaries, plot corners, coordinate baselines. Boldness lives in one motif; everything else stays calm Material 3 so the app reads as an official record, not a consumer app.

05 — MOTION

Lightweight, and it disappears on request

Every motion below is opacity/transform only (cheap on Flutter), 150–300ms, and wrapped so MediaQuery.disableAnimations / reduce-motion collapses it to an instant state change — no dependencies.

Entrance reveal
Screen content fades + rises 8dp. 220ms · easeOutCubic. Reduce-motion → appears instantly.
Grid cascade
Home tiles stagger 40ms each on first paint. ≤240ms total. Off → all at once.
Press-to-scale
Tiles & buttons scale to 96% while pressed. 120ms. Off → color-only feedback.
Signature: baseline draw
Tracking survey line animates its fill L→R (or R→L in RTL) once. 300ms. Off → static fill.
Cart / total bump
Counter chip & total scale-pulse to 1.06 on change. 160ms. Off → value swaps.
Page transition
Shared-axis / gentle zoom between screens. RTL flips direction. Off → cross-fade or cut.
Handoff note. All values above are literal: hexes are the fixed brand set, spacing is the 4dp scale, radii 8/12/16/999, type is IBM Plex (Sans / Sans Arabic / Mono). Drop them into app_theme.dart as a ColorScheme + TextTheme, and each component in §03 maps to one shared widget.
